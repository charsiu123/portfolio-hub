### Habit PWA

オフラインでも使える習慣タイマーです。端末内に記録を保存し、manifest と Service Worker によるインストール対応を実装しています。
HTML / CSS / JavaScript / Service Worker

### Preset Mall

音楽プリセットを探し、作り手へ進むための体験を検証中のクライアント側MVPです。検索・絞り込みと導線の試作を実演できます。
Next.js / React / TypeScript / Tailwind

## 公開範囲

このリポジトリとデモには、個人情報・認証情報・配布教材・決済やアカウント登録などの本番機能は含めていません。
