# Private CS Portfolio

A recruiter-facing index for selected personal software projects.

## Included work

- Habit PWA — local-first offline habit tracking.
- Study Habit — local study planning, timer, and history demo.
- Preset Mall — a clearly labeled client-side marketplace MVP mock.

## Privacy and security

- This site is intended for invitation-only access.
- It contains no personal data, course-provided source code, payment flow, authentication flow, analytics, database, upload endpoint, or API credential.
- Hosted responses include `noindex` and restrictive static security headers in `public/_headers`.
- Live links remain disabled until access control is configured.
